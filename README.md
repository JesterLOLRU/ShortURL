# Сервис по сокращению ссылок

## Технический стек
* Java 11
* Spring Boot
* Maven
* PostgreSql
* Liquibase
* Swagger

## Запуск
В папке проекта:
> mvn compile

> mvn clean package 

> docker build -t url:project .

> docker run -p 8000:8000 url:project .


## Endpoints
Потыкать все можно здесь:
http://localhost:8080/swagger-ui.html
